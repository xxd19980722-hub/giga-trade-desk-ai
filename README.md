# giga-trace-desk-ai

一个面向广告赛马分析的 AI 工作台 MVP。

当前目标：
- 定义账户组
- 查询渠道/投放报表
- 做多组对比
- 做素材分析
- 生成分析报告

## 当前目录结构

- `docs/`：产品、架构、数据结构文档
- `skills/`：技能定义与示例
- `mock/`：本地 mock 数据
- `frontend/`：前端工作台
- `backend/`：后端服务与编排层
