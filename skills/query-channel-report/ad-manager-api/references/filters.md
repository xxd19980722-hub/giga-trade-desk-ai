| 9 | 信息流3.0 |
| 10 | PUSH |
| 11 | 场景化投放 |

### 各平台支持的子渠道

**应用商店类平台**（OPPO=14, 荣耀=91, 小米=98, VIVO=99, 华为非商店=4001, 华为商店=4002）：
- 1=商店, 2=非商店, 3=CPT, 10=PUSH, 12=非商店主包

**今日头条**(1)：信息流(4), 直播(5), 品牌(7)

**快手**(2)：信息流(4), 直播(5), 达人商推(8), 品牌(7)

**广点通**(3)：信息流(4), 直播(5), 信息流3.0(9), 品牌(7), 场景化投放(11)

**其余平台**（默认）：信息流(4), 直播(5)
